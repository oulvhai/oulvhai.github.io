MWeb 2.0.9 版发布！修正 macOS 的问题和发布到 evernote 重大功能！

2016/08/25  posted in 使用文档  版本发布 

这次发布主要是修正 macOS 打开文档后空白的问题，另外还有一个比较重大的功能，就是：

  1. 发布到 evernote 的样式，终于跟选择的预览样式一样了！另外，还支持自定义样式！
  2. 发布到 evernote 支持代码高亮了！
  3. 发布到 evernote LaTeX 和 TOC 也会显示了！
  4. 发布到 evernote 时，也会把 ECharts、mermaid、PlantUML 等画图库生成的图片发布到 evernote 了，这个功能要用到 自定义预览样式，让 MWeb 支持 ECharts、mermaid、PlantUML 等任何 JS 画图库 这篇文章里的预览样式，老用户也要去下载新的样式才能支持 evernote。

需要重点说明的是，这个功能是由 cyhhao 这位朋友帮忙做的！而且这个功能所用的代码已经开源，网址为：https://github.com/cyhhao/html2enml。这个库的主要功能是把 html 转化为印象笔记所支持的 enml 格式。

另外的改进还有：

  • 由于编辑器内的连结设置为可点时，再想编辑比较麻烦，现在增加：按住 CMD 键再点击时，将不会打开连结。
  • 修正 把 HTML 粘贴为 Markdown 时，代码块中如果有 < 和 > 字符，会不正常的问题。修正斜体会转化为下划线的问题。
  • 修正 mweb-otcpress 预览样式中，二级标题有时会被切掉一些的问题。

另外要说明的是：由于不确定这次针对 macOS 的 bug 修正是否稳定（我测试是没有问题的），所以 Mac App Store 版本会过几天才提交。
