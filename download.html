MWeb 最近更新情况

2021 年 MWeb 更新情况

  • 1/9: 已发布 MWeb 的新编辑器的单独 App： macOS 版 One Markdown。
  • 1/29: 已发布 iOS/iPadOS 版 One Markdown。
  • 2/2: 决定先进行把新编辑器整合进 MWeb 的工作，主要工作有：
    1. 把新编辑器整合进 MWeb。
    2. macOS/iOS/iPadOS UI 改进，将采用更现代的界面。
    3. 改进同步功能，将支持 CloudKit 同步。
  • 4/6: MWeb 4.0.0 for macOS 测试版发布。

感觉这可能算是 MWeb 4.0 版了吧，重点说明： 此更新是免费的。

MWeb 3.4.1 更新情况

MWeb 3.3.9 更新情况

  • 增加：插入当前日期的快捷健（CMD + D），插入当前日期和时间的快捷键（CMD + Shift + D）。也可以使用菜单：标签 - 插入当前时间。
  • 增加：目前可以设置暗主题的预览样式了，在 “偏好设置 - 主题&样式” 中设置。
  • 增加：静态网站功能中，当在模板中直接输出数组或者字典变量时，会直接变成 json 输出。大概用法为 <script>var postsArr = {{posts}}; var catsArr = {{p.cats}};</script> 。这个是为了解决模板语言描述能力不足问题，遇到这个问题，直接把变量变成 js 对象，这样就可以直接使用 js 输出了。
  • 修正发布到 ghost 之后，不能再更新到 ghost 的问题。
  • 修正上传到自定义图床时，POST 数据格式不以 -- 结束的问题。
  • 修正在 macOS 10.15.3 及以上版本中，亮暗主题切换不正确问题。
  • 其他 bug 修正及改进。

MWeb 3.2.8 及以下版本升级到 3.3.1 版本说明

这个说明只针对非 Mac App Store 版本，Mac App Store 版本不受影响。

重要说明：

3.2.8 及以下版本的 MWeb 要更新到 3.3.1 及以上版本时，请 下载 或更新到 3.2.8 版本，然后使用菜单： MWeb - 导出偏好设置数据... 功能，导出所有设置。然后再升级到 3.3.1 版本。当升级到 3.3.1 版本后，请使用菜单： MWeb - 导入偏好设置数据... 功能，导入 3.2.8 版本导出的偏好设置数据！

提示：如果你没有导出偏好设置数据就更新到了 3.3.1，可以重新下载 3.2.8 版本，再导出，然后再在 3.3.1 版本导入即可。
