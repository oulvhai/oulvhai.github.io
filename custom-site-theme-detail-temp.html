Site theme floder detail

The floder name is the Theme name, the floder contain below document and floder:

  • asset – this floder contain all asset,when build site,asset floder will copy to the site output location.
  • atom.xml – site feed template
  • header.html – site header
  • footer.html – site footer
  • archives.html – archives page template
  • category.html – all categories page template
  • sidebar.html – sidebar template
  • page.html – page template, if the document Is page,then use this template
  • post.html – article template
